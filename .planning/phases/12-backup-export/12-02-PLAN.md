**In rust/src/api/mpc_engine.rs:**

1. Add imports at top: `use k256::{ProjectivePoint, Scalar, NonZeroScalar, AffinePoint};` and `use k256::elliptic_curve::ops::Invert;` and `use crate::session::EXPORTED_KEYS;`.

2. Add private helper struct (inside mpc_engine.rs, not public):
```rust
#[derive(serde::Deserialize)]
struct KeyshareExportFields {
    s_i: Scalar,
    x_i_list: Vec<NonZeroScalar>,
    rank_list: Vec<u8>,
    party_id: u8,
    public_key: AffinePoint,
}
```

3. Add private helper:
```rust
fn lagrange_coeff_2of2(x_i: &Scalar, x_j: &Scalar) -> Result<Scalar, String> {
    let diff = x_j - x_i;
    let diff_inv = Option::<Scalar>::from(diff.invert())
        .ok_or("degenerate Lagrange: x_i == x_j")?;
    Ok(x_j * diff_inv)
}
```

4. Replace the `export_private_key` stub with full implementation:
   - Parse both JSON strings into `KeyshareExportFields`
   - Validate both shares have `rank_list` all zeros (2-of-2 standard Lagrange)
   - Validate `fields_0.public_key == fields_1.public_key` (same DKG group)
   - Extract `x_0 = Scalar::from(fields_0.x_i_list[fields_0.party_id as usize])`, same for party 1 (cast NonZeroScalar to Scalar via `*x_i`)
   - Compute `lambda_0 = lagrange_coeff_2of2(&x_0, &x_1)?`
   - Compute `lambda_1 = lagrange_coeff_2of2(&x_1, &x_0)?`
   - `let private_key = lambda_0 * fields_0.s_i + lambda_1 * fields_1.s_i;`
   - Verify: `let derived_pub = (ProjectivePoint::GENERATOR * private_key).to_affine();`
   - Assert `derived_pub == fields_0.public_key`, return Err if mismatch ("private key reconstruction failed: public key mismatch")
   - Derive EVM address from uncompressed public key bytes: `derived_pub.to_encoded_point(false)` -> `derive_evm_address(point.as_bytes())?`
   - Convert private_key to hex: use `k256::elliptic_curve::ScalarPrimitive::<k256::Secp256k1>::from(&private_key)` then `hex::encode(scalar_primitive.to_bytes())`
   - Register exported key: `EXPORTED_KEYS.lock().unwrap().insert(hex::encode(fields_0.public_key.to_encoded_point(true).as_bytes()));`
   - Return `ExportResult { private_key: hex_key, address, exported: true }` serialized as JSON

5. Add EXPORTED_KEYS check in `sign_start` (after keyshare deserialization, around line 709):
   - After `let public_key = keyshare.public_key;` add:
   ```rust
   let pk_hex = hex::encode(public_key.to_encoded_point(true).as_bytes());
   if EXPORTED_KEYS.lock().unwrap().contains(&pk_hex) {
       return Err("signing rejected: keyshare has been exported".to_string());
   }
   ```

NOTE on Scalar to hex conversion: `k256::Scalar` does not directly expose bytes. Use `ScalarPrimitive::from(&private_key)` then `.to_bytes()` which gives `FieldBytes` (GenericArray<u8, U32>). Then `hex::encode(&to_bytes)`.

| export_private_key -> ExportResult.private_key | Full private key in memory as hex String |
| EXPORTED_KEYS -> sign_start | Runtime guard prevents signing with exported keyshare |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-03 | Info Disclosure | ExportResult.private_key | accept | Private key hex is the intended output; caller (Dart layer) responsible for secure handling and zeroization |
| T-12-04 | Repudiation | sign after export | mitigate | EXPORTED_KEYS HashSet checked at sign_start entry; blocks signing with exported keyshare's public key |
| T-12-05 | Tampering | mismatched keyshare JSONs | mitigate | public_key equality check between both shares; G*reconstructed_key == public_key verification |
| T-12-06 | Denial of Service | EXPORTED_KEYS memory growth | accept | Only grows by one entry per export call; bounded by number of wallets; negligible memory |
| T-12-07 | Info Disclosure | server_share_private transport | accept | Out of Rust scope; caller must ensure secure channel (documented in RESEARCH.md A2) |
